Lining.js

Write css style for line in typography Like this: 
<div data-lining>
    First line.<br/>
    Second line.<br/>
    Last line.
</div>
<style>
    line { font-size:16px; }
    line[first] { font-size:2em; font-weight: bold; }
    line[i="2"] { font-size:1.5em; }
</style>
<script src="http://nodejs.in/lining.js/build/lining.min.js"></script>
